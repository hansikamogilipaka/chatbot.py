# 🤖 Rule-Based AI Chatbot

## 📌 Overview
This project is a simple rule-based chatbot built in Python.  
It can run in the terminal (`chatbot.py`) or in a separate GUI window (`chatbot_gui.py` using Tkinter).

## ⚙️ Features
- Responds to greetings and common questions
- Shows current time and date
- GUI interface with scrollable chat window
- Exit commands (`bye`, `exit`, `quit`)

## 🚀 How to Run
### Terminal Version
```bash
python chatbot.py
